# 図書館

* 値オブジェクト (Value Object)
  * 書籍 (Book)
    * 作品としての本（物理的な実体ではない）
    * ID (ISBN等) とタイトルを持つ

* エンティティ (Entity)
  * 蔵書 (BookItem)
    * 図書館が所持する物理的な本
    * 特定の書籍カタログ (Book) に紐づく
    * 状態 (State) を持つ
      * 在庫あり (In Stock)
      * 貸出中 (Borrowed)
  * 利用者 (Borrower)
    * 本を借りる人
    * 借りている蔵書 (BookItem) を管理する
    * ルール
      * 貸出中 (Borrowed) の蔵書は借りられない
      * 既に借りている蔵書は借りられない
  * 貸し出し (Loan)
    * 蔵書と利用者を結びつける
    * 貸出日、返却期限、返却日を持つ

* 集約 (Aggregate)
  * 在庫 (Stock)
    * 図書館全体の蔵書 (BookItem) を管理する集約ルート
    * 蔵書の追加や、利用可能な蔵書の検索を行う
